# Design Reference Library — 品牌设计系统参考库

基于 [awesome-design-md](https://github.com/VoltAgent/awesome-design-md) 的 58 个顶级品牌设计系统参考。

## 使用方式

当需要参考某个品牌的设计系统时，使用以下方式获取：

### 方式 1：在线获取（推荐，获取最新版本）

```bash
# 获取单个品牌的 DESIGN.md
curl -sL "https://raw.githubusercontent.com/VoltAgent/awesome-design-md/main/design-md/{brand}.md"

# 示例：获取 Stripe 的设计系统
curl -sL "https://raw.githubusercontent.com/VoltAgent/awesome-design-md/main/design-md/stripe.md"
```

### 方式 2：使用 npx CLI

```bash
npx getdesign@latest add <brand>
```

### 方式 3：读取本地缓存

如果已下载到 `references/` 目录：
```
skills/design-reference/references/{brand}.md
```
