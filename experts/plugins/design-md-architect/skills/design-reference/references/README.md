# 品牌参考索引

本目录存放从 awesome-design-md 仓库获取的品牌 DESIGN.md 参考文件。

## 获取方式

品牌参考文件通过在线方式实时获取（确保最新版本）：

```bash
curl -sL "https://raw.githubusercontent.com/VoltAgent/awesome-design-md/main/design-md/{brand}.md"
```

## 文件格式

每个品牌 DESIGN.md 遵循统一的 9 章节结构（约 280-350 行）：

1. Visual Theme & Atmosphere
2. Color Palette & Roles
3. Typography Rules
4. Component Stylings
5. Layout Principles
6. Depth & Elevation
7. Do's and Don'ts
8. Responsive Behavior
9. Agent Prompt Guide
