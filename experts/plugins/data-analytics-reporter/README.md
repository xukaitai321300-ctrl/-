# 数妙妙 - 数据分析报告师

将复杂数据转化为战略洞察，提供指标诊断、KPI框架设计、数据质量评估与决策报告。

## 核心能力

| 技能 | 描述 |
|------|------|
| `data-quality-assessment` | 数据质量评估（粒度、新鲜度、空值、重复、Schema漂移等）与分析验证（方法论、计算、可视化QA） |
| `metric-diagnostics` | 指标变动诊断（归因分解、驱动验证）与面向领导层的KPI汇报（WBR/MBR/QBR） |
| `kpi-design` | KPI框架设计（候选生成→比选→目标设定）与市场规模估算（TAM/SAM/SOM） |
| `data-visualization` | 数据可视化设计规范（图表选择、调色板策略、质量标准） |
| `analytics-reporting` | 分析报告构建、仪表盘设计与产品/业务分析 |

## 使用场景

- 数据质量检查与分析验证
- 核心指标下降/异常的根因诊断
- KPI框架设计与目标设定
- 面向管理层的定期业务回顾（周报/月报/季报）
- 产品/业务决策的数据支持分析
- 市场规模和机会评估
- 仪表盘设计与报告生成
