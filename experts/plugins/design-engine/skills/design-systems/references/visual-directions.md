# 5 大视觉方向（Visual Directions）

5 种经过验证的视觉学派，每种都有确定性的色彩方案和字体栈，避免 AI 随机选择导致的平庸结果。

---

## 1. Editorial Monocle（编辑质感）

**哲学**：像高端杂志一样排版——每一行都有呼吸感，每一个元素都经过深思熟虑。

| 维度 | 定义 |
|------|------|
| 字体 | 衬线展示字体 + 无衬线正文（如 Playfair Display + Source Sans Pro） |
| 色彩 | 低饱和度、大面积留白、1-2 个点缀色 |
| 布局 | 非对称网格、大留白、错落感 |
| 元素 | 细线分隔、小型大写字母、拉丁引号标记 |
| 适用 | 品牌展示、产品故事页、创意机构、时尚/奢侈品 |

**OKLCh 调色板**：
- 主色：oklch(30% 0.02 60) — 近黑暖灰
- 背景：oklch(98% 0.005 80) — 暖白
- 点缀：oklch(55% 0.15 30) — 铜棕色

---

## 2. Modern Minimal（现代极简）

**哲学**：少即是多——去掉一切不必要的装饰，让内容和功能说话。

| 维度 | 定义 |
|------|------|
| 字体 | 几何无衬线体（如 Inter, Satoshi, General Sans） |
| 色彩 | 黑白灰为主，一个强调色 |
| 布局 | 严格栅格、等距、对齐 |
| 元素 | 大标题、卡片式内容、清晰的层级 |
| 适用 | SaaS 产品、开发者工具、效率工具 |

**OKLCh 调色板**：
- 主色：oklch(25% 0 0) — 纯黑
- 背景：oklch(100% 0 0) — 纯白
- 强调：oklch(60% 0.20 250) — 蓝色

---

## 3. Tech Utility（技术工具感）

**哲学**：功能至上——像仪表盘一样信息密集，像终端一样精确。

| 维度 | 定义 |
|------|------|
| 字体 | 等宽或半等宽字体（如 JetBrains Mono, IBM Plex Mono） |
| 色彩 | 深色背景、霓虹强调色、高对比 |
| 布局 | 高信息密度、紧凑间距、多列 |
| 元素 | 数据表格、状态指示器、代码块风格 |
| 适用 | 开发者仪表盘、数据平台、DevOps 工具 |

**OKLCh 调色板**：
- 背景：oklch(15% 0.01 260) — 深蓝黑
- 文本：oklch(90% 0.01 100) — 浅灰绿
- 强调：oklch(75% 0.20 160) — 霓虹绿

---

## 4. Brutalist（粗犷主义）

**哲学**：打破规则——用不完美和原始感创造记忆点。

| 维度 | 定义 |
|------|------|
| 字体 | 超粗标题字体 + 系统等宽（如 Space Grotesk Black + monospace） |
| 色彩 | 高对比、纯色块、可以大胆撞色 |
| 布局 | 打破栅格、重叠、溢出 |
| 元素 | 粗边框、大字号、生硬的直角、无装饰 |
| 适用 | 创意机构、实验性项目、个人作品集、潮牌 |

**OKLCh 调色板**：
- 背景：oklch(95% 0 0) — 粗纸白
- 主色：oklch(20% 0 0) — 浓黑
- 强调：oklch(70% 0.25 30) — 橙红

---

## 5. Soft Warm（柔和温暖）

**哲学**：亲和力——圆润的形状、温暖的色调、让人放松的视觉节奏。

| 维度 | 定义 |
|------|------|
| 字体 | 圆润无衬线体（如 Nunito, Quicksand, Poppins） |
| 色彩 | 暖色调、低对比度、柔和渐变（允许微妙渐变） |
| 布局 | 宽松间距、大圆角、卡片式 |
| 元素 | 柔和阴影、圆角卡片、插图/图标丰富 |
| 适用 | C 端消费品、教育产品、健康/生活方式、社交产品 |

**OKLCh 调色板**：
- 背景：oklch(97% 0.01 80) — 暖米白
- 主色：oklch(60% 0.18 20) — 珊瑚粉
- 辅色：oklch(70% 0.12 150) — 薄荷绿

## 选择指南

| 用户需求关键词 | 推荐方向 |
|--------------|---------|
| 高端、品牌故事、杂志感 | Editorial Monocle |
| 简洁、SaaS、工具、效率 | Modern Minimal |
| 数据密集、开发者、暗色 | Tech Utility |
| 创意、实验、个性、潮 | Brutalist |
| 温馨、亲和、消费品、社交 | Soft Warm |
