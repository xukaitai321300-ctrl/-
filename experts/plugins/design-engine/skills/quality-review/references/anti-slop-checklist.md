# Anti-Slop 完整检测清单

"Slop" 指 AI 生成的低质量、千篇一律、缺乏思考的设计模式。本清单定义了所有需要检测和避免的 AI 设计反模式。

---

## P0 — 必须修复（阻断发布）

出现任何一条即不可发布：

### 视觉反模式
- [ ] **紫色/彩虹渐变背景** — AI 最爱的"高级感"捷径，实际是最廉价的视觉
- [ ] **通用 emoji 作为图标** — 如用 🚀 表示速度、💡 表示想法
- [ ] **圆角卡片 + 左侧彩色边框** — AI 套路万金油组件
- [ ] **手绘风 SVG 人物插图** — generic AI illustration style
- [ ] **深色背景 + 霓虹渐变文字** — 过时的"科技感"

### 内容反模式
- [ ] **编造统计数据** — "提升 300% 效率"、"节省 10000 小时"等无来源数字
- [ ] **虚假社会证明** — 杜撰的用户评价、公司 logo 墙
- [ ] **空洞的形容词堆砌** — "革命性的"、"颠覆式的"、"下一代"

### 技术反模式
- [ ] **明显破碎的布局** — 溢出、重叠、错位
- [ ] **文本对比度不达标** — WCAG AA 不通过（正文 < 4.5:1, 大标题 < 3:1）
- [ ] **完全无响应式** — 移动端不可用
- [ ] **依赖外部资源但未加载** — 404 的图片、未加载的字体

---

## P1 — 建议修复（影响品质）

累计 3 条以上建议打回修正：

### 排版问题
- [ ] **Inter 作为展示字体** — 作为 body font 可以，作为 display 太平庸
- [ ] **段落宽度超过 75 字符** — 阅读体验差
- [ ] **行高不足** — body text line-height < 1.5
- [ ] **字号层级不清晰** — H1/H2/H3 视觉差异不够

### 色彩问题
- [ ] **同一页面 4+ 种不同色调** — 视觉混乱
- [ ] **纯黑 #000000 作为文本色** — 太刺眼，应用深灰
- [ ] **色彩使用无语义** — 红/绿色用于非状态用途

### 布局问题
- [ ] **留白不足** — 内容过于拥挤
- [ ] **卡片/元素阴影堆叠过多** — 超过 2 层深度
- [ ] **CTA 按钮过多** — 一屏内超过 2 个主 CTA
- [ ] **Hero 区域内容过载** — 标题 + 副标题 + 描述 + 2个按钮 + 图片全塞一起

### 交互问题
- [ ] **动画时长 > 300ms 且无 reduced-motion** — 可访问性问题
- [ ] **无 hover/focus 状态** — 交互元素无反馈
- [ ] **链接无法区分** — 与普通文本视觉相同

---

## P2 — 可选优化（锦上添花）

不影响发布，但做了会更好：

- [ ] **图标大小不统一** — 视觉节奏可优化
- [ ] **没有自定义滚动条** — 深色主题下默认滚动条突兀
- [ ] **缺少 loading 状态设计** — skeleton screen 或 spinner
- [ ] **图片缺少 alt 属性** — 无障碍访问
- [ ] **没有 404/空状态设计** — 异常情况未考虑
- [ ] **字间距 (letter-spacing) 可优化** — 大标题可适当增加
- [ ] **缺少 focus-visible 样式** — 键盘导航体验

---

## Slop 替代方案

| Slop 模式 | 品牌级替代 |
|-----------|-----------|
| 紫色渐变背景 | 单色或微妙的色温变化 |
| emoji 图标 | SVG 线性图标 或 Lucide/Heroicons |
| 编造数据 | `—` 占位符或标注灰色块 |
| 手绘人物 | 抽象几何图形或真实照片 |
| 圆角卡片+左边框 | 无边框布局或极细线分隔 |
| "革命性" 文案 | 具体的功能描述或数据 |
| Inter 展示字 | 根据调性选择特征字体 |

---

## 检测执行方式

1. **先整体扫视**：打开页面，3 秒内的第一印象——有没有"一看就是 AI 做的"感觉
2. **逐项 P0 检查**：如果有 P0，直接打回
3. **逐项 P1 检查**：累计问题数
4. **P2 记录但不计入**：作为优化建议附上
5. **生成报告**：包含每个问题的具体位置和修复方案
