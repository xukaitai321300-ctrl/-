# 智数分析专家团（AI Data Copilot）

AI原生的智能数据分析专家团队，提供从数据接入到报告交付的端到端分析能力。

## 类型

Team 型（6人专家团队）

## 功能

智数分析专家团是一支由6位AI专业角色组成的数据分析精英团队。具备以下核心能力：

- **自然语言转SQL**：将口语化的数据问题自动转化为精确的SQL查询
- **Python数据科学**：统计分析、机器学习建模、数据清洗与特征工程
- **RAG知识检索**：从文档中智能检索信息，提供有来源引用的回答
- **可视化设计**：交互式图表、数据仪表盘、数据画像展示
- **报告生成**：结构化分析报告、商业洞察摘要、HTML富文本报告
- **多源数据接入**：MySQL/PostgreSQL/SQLite/ClickHouse/CSV/Excel/文档

## 标准工作流程

```
Phase 1: 需求理解与任务规划（主理人）
    ↓
Phase 2: 数据接入与查询（SQL分析师）
    ↓
Phase 3: 数据科学分析（数据科学家）
    ↓
Phase 4: 知识检索增强（知识研究员，按需）
    ↓
Phase 5: 可视化设计（仪表盘设计师）
    ↓
Phase 6: 报告生成与交付（报告撰写师）
    ↓
Phase 7: 综合交付（主理人）
```
