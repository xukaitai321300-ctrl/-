# OpenClaw 成长追踪器

> 不是冷冰冰的数据，是你的进化地图。

## 核心理念

**一切皆可量化，让成长看得见。**

记录你在 OpenClaw 上的每一次对话、任务、突破和坚持，用数据呈现真实的成长轨迹。

---

## 🚀 快速开始

说「**启动成长追踪**」初始化数据文件，之后每次对话我会自动记录你的使用数据。

## 🎯 能力维度说明

能力评分由多个因素综合计算：

| 能力 | 计算方式 |
|------|---------|
| **写作** | 文章产出数量、质量评分、字数统计 |
| **代码** | 代码任务完成数、bug 修复数、工具使用 |
| **研究** | 新话题探索数量、信息检索深度 |
| **规划** | 计划制定、任务拆解、项目管理 |
| **分析** | 问题诊断、逻辑推理、数据洞察 |
| **沟通** | 需求表达清晰度、反馈采纳率 |

## 🌡️ 状态指示

根据当日数据动态给出状态标签：

| 状态 | 条件 |
|------|------|
| 🔥 巅峰 | 所有维度提升 > 10% |
| 📈 成长期 | 整体提升 > 5% |
| 🌊 平稳期 | 变化 < 5% |
| 🔄 调整期 | 部分维度下降 |
| 🌙 低谷期 | 整体下降 > 5% |

## ⚙️ 自定义配置

在 `~/.openclaw/config.json` 中自定义：

```json
{
  "skills": {
    "openclaw-growth-tracker": {
      "name": "大表哥",
      "report_detail": "full",
      "auto_track": true,
      "weekly_report_day": "Sunday",
      "show_trends": true,
      "retention_days": 90
    }
  }
}
```
