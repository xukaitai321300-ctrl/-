class PromptOptimizer:
    def optimize_prompt(self, task_type, base_prompt, negative_prompt, case_db):
        similar_cases = case_db.find_similar_success(task_type, {})
        if not similar_cases:
            return base_prompt, negative_prompt, "No similar cases found"
        best_prompts = [c["prompt_used"]["positive"] for c in similar_cases[:3]]
        optimized = self.merge_prompts(best_prompts)
        final = self.blend_prompts(base_prompt, optimized, weight=0.7)
        return final, negative_prompt, f"Optimized based on {len(similar_cases)} cases"
    def merge_prompts(self, prompts):
        keyword_counts = {}
        for prompt in prompts:
            keywords = [k.strip() for k in prompt.split(",")]
            for kw in keywords:
                keyword_counts[kw] = keyword_counts.get(kw, 0) + 1
        merged = [kw for kw, cnt in keyword_counts.items() if cnt >= 2]
        return ", ".join(merged)

def assess_multimodal_quality(
    generated_image,
    text_input,
    image_input,
    quality_threshold=7.0
):
    """Assess quality of generated image (combined text + image)"""
    
    # 1. Image similarity (generated vs. reference image)
    if image_input:
        image_similarity = calculate_image_similarity(
            generated_image,
            image_input["source"],
            method="clip"  // Or "ssim" / "lpips"
        )
    else:
        image_similarity = 1.0  // No image input, skip
    
    # 2. Text matching (generated image vs. text prompt)
    text_matching = calculate_text_image_matching(
        generated_image,
        text_input.get("prompt", ""),
        method="clip"  // Or "blip" / "git"
    )
    
    # 3. Combined quality score (weighted average)
    quality_score = (
        0.4 * image_similarity +  // 40% weight to image similarity
        0.6 * text_matching     // 60% weight to text matching
    )
    
    # 4. Check if meets threshold
    if quality_score >= quality_threshold:
        status = "success"
        reason = f"Quality score {quality_score:.2f} >= threshold {quality_threshold}"
    else:
        status = "failed"
        reason = f"Quality score {quality_score:.2f} < threshold {quality_threshold}"
    
要点：
- return {
- "status": status,
- "quality_score": quality_score,
- "image_similarity": image_similarity,
- "text_matching": text_matching,
- "reason": reason,
- "details": {
- "image_similarity_method": "clip",
- "text_matching_method": "clip",
- "quality_threshold": quality_threshold
- }
- }

## Phase 6: 品牌设计自动化与风格库管理 (NEW in v7.2)

### 6.1 品牌设计自动化流程

**自动化设计目标**：
- 品牌VI系统自动生成（Logo、色彩、字体、辅助图形）
- 产品系列风格统一（色彩方案、材质选择、表面处理）
- Prompt素材池自动构建（根据品牌风格生成Prompt）
- 品牌宣传物料自动生成（海报、画册、展板）

**自动化设计流程**：
```
1. 接收品牌设计任务（从🐭鼠/用户）
2. 识别品牌定位和设计风格
3. 调用风格库匹配最佳风格方案
4. 生成品牌VI系统草案
5. 调用🐑羊进行AI生图（品牌应用效果预览）
6. 生成品牌设计报告
7. 存储到品牌风格库
8. 通知🐭鼠设计完成
```

**设计优先级**：
- 品牌VI系统：最高优先级（品牌基础）
- 产品系列风格：高优先级（产品一致性）
- Prompt素材池：中优先级（提升生图质量）
- 宣传物料：低优先级（可批量生成）

### 6.2 风格库自动管理与更新

**风格库分类**：
- 商务简约风（墨影/月白/棠色）
- 科技未来风（蓝色/银色/透明）
- 自然环保风（绿色/棕色/原木）
- 轻奢时尚风（金色/黑色/皮革）

**风格库内容**：
| 风格 | 色彩方案 | 材质选择 | 表面处理 | 适用产品 |
|------|----------|----------|----------|----------|
| **商务简约** | 黑/白/灰 | 不锈钢/玻璃 | 拉丝/喷砂 | 保温杯/办公杯 |
| **科技未来** | 蓝/银/透明 | 铝合金/PC | 阳极氧化/UV | 电动轮椅/智能杯 |
| **自然环保** | 绿/棕/原木 | 竹/陶瓷/硅胶 | 原纹/哑光 | 户外杯/儿童杯 |
| **轻奢时尚** | 金/黑/皮革 | 不锈钢/皮革 | 镜面/压印 | 礼品杯/限量版 |

**自动更新流程**：
```
1. 每次品牌设计完成后 → 提取风格特征
2. 匹配风格库（相似度>80% → 更新现有风格；否则 → 创建新风格）
3. 更新风格库（色彩方案、材质选择、表面处理）
4. 生成风格库更新报告
5. 通知🐭鼠风格库已更新
```

### 6.3 与十四生肖团其他Agent的联动

```
🐷生成品牌风格 → 调用🐑生成品牌应用效果预览
```

```
🐷完成品牌设计 → 调用🐔进行设计评审
→ 提供品牌标准给🐔（评审基准）
→ 🐔评审完成 → 🐷对比评审结果与品牌标准
→ 生成品牌合规性报告
```

**联动链3：🐷→🐶（品牌设计→成本分析）**

```
🐷确定品牌设计方案 → 调用🐶进行成本分析

→ 🐶分析完成 → 🐷对比成本与品牌定位
→ 生成品牌+成本综合优化建议
```

### 6.4 Prompt素材池自动构建

**素材池内容**：
- 品牌风格关键词（商务简约/科技未来/自然环保/轻奢时尚）
- 产品类型关键词（保温杯/电动轮椅/办公杯）
- 材质关键词（不锈钢/玻璃/铝合金/竹/陶瓷）
- 色彩关键词（墨影/月白/棠色/蓝色/绿色）
- 表面处理关键词（拉丝/喷砂/阳极氧化/原纹）

**自动构建流程**：
```python
# 根据品牌风格自动生成Prompt素材
def build_prompt_pool(brand_style):
    prompt_pool = {
        "style_keywords": extract_style_keywords(brand_style),
        "product_keywords": get_product_keywords(),
        "material_keywords": extract_material_keywords(brand_style),
        "color_keywords": extract_color_keywords(brand_style),
        "finish_keywords": extract_finish_keywords(brand_style)
    }
    
    # 保存到Prompt素材池
    save_to_prompt_pool(prompt_pool)
    
    return prompt_pool
```

**素材池查询**：
- 根据品牌风格查询关键词
- 根据产品类型查询推荐素材
- 根据材质/色彩/表面处理查询组合方案
