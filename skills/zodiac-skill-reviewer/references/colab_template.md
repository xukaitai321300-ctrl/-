# 协作接口章节模板

## 使用说明
当需要为生肖技能添加协作接口章节时，使用此模板。

---

## 模板结构

```markdown
## 协作接口

### 核心职责
{技能的核心职责描述，说明在协作流程中的角色}

### 输入来源
- **{来源技能名}** → 提供{数据类型}
- **{来源技能名}** → 提供{数据类型}

### 输出目标
- **{目标技能名}** ← 接收{输出数据类型}
- **{目标技能名}** ← 接收{输出数据类型}

### 协作流程图
```mermaid
flowchart LR
    A[输入技能] -->|{数据类型}| B[当前技能]
    B -->|{输出数据类型}| C[输出技能]
    B -->|{输出数据类型}| D[输出技能]
    
    classDef main fill:#ff9999,stroke:#333,stroke-width:2px
    classDef core fill:#99ccff,stroke:#333,stroke-width:2px
    classDef support fill:#99ff99,stroke:#333,stroke-width:2px
```

## 十二生肖团标准协作关系

### 🐭 鼠（product-researcher）- 主理人
- **输入**：直接接收用户需求
- **输出**：分配任务给所有其他技能，评审最终结果
- **核心**：任务分配者 + 结果评审者

### 🐮 牛（cost-analyst）- 成本分析
- **输入**：鼠（任务分配）、蛇（设计方案）
- **输出**：蛇（成本约束建议）、鼠（分析报告）

### 🐯 虎（competitor-analyst）- 竞品分析
- **输入**：鼠（调研任务）
- **输出**：蛇（竞品设计参考）、鼠（竞品分析报告）

### 🐰 兔（image-collector）- 图像采集
- **输入**：鼠（采集任务）、虎（图片分析需求）
- **输出**：兔（图片分析结果）、羊（生图参考）、马（生图素材）

### 🐲 龙（standards-analyst）- 行业标准
- **输入**：鼠（标准查询任务）、蛇（设计方案）
- **输出**：蛇（标准符合性检查）、鼠（标准报告）

### 🐍 蛇（product-designer）- 产品设计
- **输入**：鼠（设计任务）、虎（竞品分析）、牛（成本约束）
- **输出**：马（渲染任务）、羊（生图任务）、鸡（评审任务）、狗（成本优化）

### 🐴 马（sd-comfy-expert）- SD/ComfyUI生图
- **输入**：鼠（生图任务）、蛇（设计方案）、羊（生图指导）
- **输出**：蛇（设计效果图）、鼠（生图结果）

### 🐏 羊（ai-image-generator）- AI生图核心
- **输入**：鼠（生图任务）、蛇（设计参考）、马（生图参数）
- **输出**：蛇（生图结果）、鸡（生图评审）、鼠（最终结果）

### 🐓 鸡（design-reviewer）- 设计评审
- **输入**：鼠（评审任务）、蛇（设计方案）、羊（生图结果）
- **输出**：蛇（改进建议）、鼠（评审报告）

### 🐶 狗（skill-optimizer）- 情报收集/参数调优
- **输入**：鼠（情报任务）、蛇（设计参数优化需求）
- **输出**：蛇（参数优化建议）、马（生图参数调优）、羊（提示词优化）

### 🐷 猪（brand-designer）- 品牌设计/Prompt素材
- **输入**：鼠（品牌设计任务）
- **输出**：羊（生图风格参考）、马（生图Prompt素材）、鼠（品牌设计方案）

## Mermaid流程图样式规范

- **main**（主理人/鼠）：`fill:#ff9999`
- **core**（核心设计/蛇）：`fill:#99ccff`
- **support**（支持技能/其他）：`fill:#99ff99`

## 快速生成模板（Python）

```python
def generate_collab_section(skill_name, inputs, outputs, description):
    """生成协作接口章节Markdown"""
    md = f"## 协作接口\n\n"
    md += f"### 核心职责\n{description}\n\n"
    md += "### 输入来源\n"
    for inp in inputs:
        md += f"- **{inp}** → 提供输入数据\n"
    md += "\n### 输出目标\n"
    for outp in outputs:
        md += f"- **{outp}** ← 接收输出结果\n"
    return md
```
