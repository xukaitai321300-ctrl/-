# 📋 BigPlan · AI产品调研

按产品方向，自动完成市场分析、技术评估、供应链调研，输出**高/中/低三套产品规格方案**，为产品决策提供参考。

## 触发词

`产品调研` `调研这个产品` `帮我分析` `产品方向` `想做一款` `产品规划` `market research` `product analysis`

## 工作流程

### 核心调研维度

```
你：调研智能家居网关
  ↓
① 市场调研
   · 搜索同类产品/竞品
   · 价格带与定位分析
   · 用户痛点与市场机会
   · 市场规模与趋势
  ↓
② 技术评估
   · 可行技术路线
   · 主控芯片方案对比
   · 通信协议选择
   · 认证要求
   · 软件栈评估
  ↓
③ 供应链调研
   · 核心元器件供应
   · 代工方案
   · 模具与认证费用
   · BOM 成本估算
  ↓
④ 输出三套方案
   🥉 入门版 · 最低成本快速验证
   🥈 标准版 · 主力产品平衡性能与价格
   🥇 旗舰版 · 全功能技术标杆
```

### 搜索策略

根据用户的产品描述，自动构建多角度搜索：

```python
# 例：用户说"调研一款户外储能电源"
queries = [
    "户外储能电源 产品 2025 市场分析",          # 市场
    "便携储能电源 竞品 价格 对比",               # 竞品
    "户外电源 技术方案 电芯 BMS",                 # 技术
    "便携储能 供应链 电池 代工",                   # 供应链
    "户外电源 用户评价 痛点",                      # 用户
]
```

### 三套方案输出规范

每套方案必须包含以下字段。如某些字段因数据不足无法确定，明确标注"待验证"而非留空。

| 字段 | 说明 | 示例 |
|:--|:--|:--|
| **目标用户** | 谁买、什么场景 | 户外露营人、家庭应急 |
| **关键规格** | 核心参数 | 300W/500Wh, 100W快充 |
| **硬件方案** | 主控/电芯/接口 | 33650电芯×4 + BMS |
| **软件功能** | App/控制/联网 | App远程控制 |
| **BOM成本** | 元器件材料成本 | ¥550-650 |
| **建议零售价** | 定价区间 | ¥1,299-1,699 |
| **研发周期** | 从立项到量产 | 6-8个月 |
| **研发投入** | 团队+认证+模具 | ¥20-30万 |
| **风险** | 技术和市场风险 | 电芯供应波动 |

### 输出详细度规范

| 板块 | 详细度要求 | 篇幅参考 |
|:--|:--|:-:|
| 市场概况 | 3-5 个要点，有据可查 | 100-300 字 |
| 技术可行性 | 2-3 条技术路线对比 + 理由 | 150-400 字 |
| 供应链评估 | 核心器件+成本区间 | 100-250 字 |
| **每套方案** | **9 个字段完整填写** | **200-400 字/套** |

**字段填写原则：**
- 有公开数据支撑的 → 写具体（如芯片型号、价格区间）
- 数据不足的 → 明确标注

## ⚠️ 注意事项

1. **数据精度** — BOM成本/定价/研发周期为公开市场数据估算，非工厂报价
2. **时效性** — 市场数据基于搜索结果，可能滞后 3-6 个月
3. **偏重硬件** — 当前版本更适合硬件/电子产品调研，纯软件产品建议再补充
4. **输出长度** — 一份完整报告内容较多，会分多段发送避免截断
