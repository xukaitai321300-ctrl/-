# 贡献指南

感谢您对保温杯材料与表面处理技术数据库技能包的关注！

## 如何贡献

### 报告问题

如果您发现任何问题或有改进建议，请通过以下方式联系我们：

- 创建Issue描述问题
- 提供详细的重现步骤
- 附上相关数据和截图

### 贡献数据

我们欢迎以下类型的贡献：

1. **新材料数据**：提供新型保温杯材料的技术参数
2. **新工艺数据**：补充表面处理和LOGO制作工艺
3. **成本数据更新**：提供最新的成本信息
4. **本地供应商信息**：补充永康本地供应商信息

### 贡献步骤

1. Fork本仓库
2. 创建您的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开一个Pull Request

### 数据格式要求

所有数据贡献必须遵循以下格式：

- **材料数据**：使用`data/materials.json`中的格式
- **表面处理数据**：使用`data/surface_treatment.json`中的格式
- **LOGO工艺数据**：使用`data/logo_process.json`中的格式

### 代码贡献

如果您想改进脚本或添加新功能：

1. 确保代码有详细的中文注释
2. 添加示例用法
3. 更新相关文档

## 社区行为准则

请尊重其他贡献者，保持专业和友好的交流氛围。

## 联系方式

- 维护者：速凡团队
- 位置：浙江永康
- 更新频率：每季度更新

感谢您的贡献！
