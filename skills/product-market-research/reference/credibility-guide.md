# Credibility Guide 可信度评级细则

## 评级体系

所有搜集到的信息必须经过可信度评级。评级结果直接影响该信息在报告中的权重和分析深度。

---

## ⭐⭐⭐ 高可信度 — 重点分析

### 判定标准

- 发布主体是政府机构、监管部门、官方标准化组织
- 发布主体是行业官方协会或非营利标准制定机构
- 品牌/厂商官方网站直接发布的产品规格和定价
- 学术机构发表的正式研究报告
- 含有可验证原始数据的官方统计公报

### 来源示例

| 类型 | 示例 |
|------|------|
| 政府网站 | governments (.gov / .go.jp / .go.kr / .gov.cn 等) |
| 监管部门 | 市场监管总局、FTC (美国联邦贸易委员会)、消費者庁 (日本) |
| 官方标准 | JIS 标准、CE 认证、3C 认证、ANSI 标准 |
| 官方协会 | 行业协会白皮书、TÜV/SGS 等国际认证机构报告 |
| 品牌官网 | 产品官方页面上的规格、定价、销售渠道信息 |

### 分析要求

- 这类信息应作为报告的**核心依据**
- 详细引用数据，标注具体来源和发布日期
- 如果官方信息之间存在矛盾，同时列出并分析可能原因
- 官方信息如果过时（>3 年），在报告中标注时效性警告

---

## ⭐⭐ 中可信度 — 适当分析

### 判定标准

- 知名正规媒体（有编辑团队、有事实核查流程）
- 专业产品评测机构（有标准化测试流程）
- 商业行业报告（尼尔森、Gartner、IDC 等知名机构）
- 有行业影响力的专业博客/媒体
- 第三方电商平台的聚合销售数据

### 来源示例

| 类型 | 示例 |
|------|------|
| 科技媒体 | The Verge、CNET、IT之家、価格.com |
| 评测机构 | Consumer Reports (美国)、Stiftung Warentest (德国)、What Hi-Fi |
| 行业报告 | Euromonitor、Statista、艾瑞咨询、易观 |
| 电商聚合 | 京东排行榜、Amazon Best Sellers、Lazada 热销榜 |
| 知名媒体 | 财经类、科技类正经媒体 |

### 分析要求

- 这类信息作为**补充和佐证**
- 有实际测试数据的优先采用
- 多个来源（>2 个）指向同一结论时可信度提升
- 如与官方信息矛盾，优先信赖官方信息

---

## ⭐ 低可信度 — 简要分析

### 判定标准

- 个人用户发布在社交平台/论坛/电商评论区的评价
- 无法确认作者身份和资质的博客文章
- 匿名或无署名发布的信息
- 可能存在利益冲突的内容（如付费推广）

### 来源示例

| 类型 | 示例 |
|------|------|
| 论坛/社区 | Reddit、知乎、贴吧、5ch、Clien |
| 用户评价 | Amazon 用户评论、京东评价、Coupang 评论 |
| 社交媒体 | Twitter (X)、微博、小红书、TikTok |
| 个人博客 | 无编辑团队的独立博客、个人评测频道 |
| 短视频 | YouTube、B 站、抖音的个人评测视频 |

### 分析要求

- 只提取**有一定数量共识的意见**（如 10+ 人提到同一问题）
- 不引用单一个体的极端评价
- 汇总趋势而非具体案例
- 用模糊量化词（如「多数用户认为」「部分用户反映」）
- 明确标注「以下信息来自用户评价，仅供参考」

---

## ❓ 存疑 — 仅列出，不纳入分析

### 判定标准

- 信息来源完全匿名且无法追溯
- 信息明显过时且无更新版本
- 内容带有明显广告/营销性质
- 与其他多个可信来源严重矛盾
- AI 生成内容嫌疑明显的页面

### 处理方式

- 在来源链接中列出，标注「❓ 来源存疑，未纳入分析」
- 不引用其数据
- 如果这是唯一能找到的信息，在报告中明确告知用户

---

## 评级决策流程

```
找到一条信息
  │
  ├── 发布者是政府/官方/品牌官方？
  │   ├── 是 → ⭐⭐⭐
  │   └── 否 ↓
  │
  ├── 发布者是有编辑团队的知名媒体/评测机构？
  │   ├── 是 → ⭐⭐
  │   └── 否 ↓
  │
  ├── 发布者是可识别的个人（有公开身份）？
  │   ├── 是 → ⭐
  │   └── 否 ↓
  │
  └── 来源完全匿名/无法验证 → ❓
```

## 特殊情况

### 电商平台评分/销量数据
- 官方旗舰店标价 → ⭐⭐⭐
- 第三方卖家标价 → ⭐⭐
- 用户评论内容 → ⭐
- 刷单/虚假评价嫌疑 → ❓

### 新闻媒体报道
- 知名国际/国内正规媒体 → ⭐⭐
- 地方小媒体、自媒体 → ⭐
- 内容农场、AI 生成内容 → ❓

### 社群信息
- 有实名认证的行业专家发言 → ⭐⭐
- 活跃社群中多数人的共识 → ⭐
- 匿名单次发言 → ❓
