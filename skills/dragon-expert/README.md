# Dragon Expert Skill Package (十二生肖团 - 龙专家)#

龙专家（竞品分析专家）技能包，负责深度解构竞品生图方案。包含竞品分析、竞品工艺分析、竞品设计风格分析等功能。是十二生肖团的竞品分析者。

## Features#

- **Competitive Product Analysis**: Product analysis, craftsmanship analysis, design style analysis
- **Image Generation Solution Deconstruction**: Prompt analysis, workflow analysis, parameter analysis
- **Competitive Comparason**: Multi-dimensional comparison with competitor products#

## Installation#

1. Ensure Python 3.8+ is installed.
2. No additional Python dependencies are required (only standard libraries: json, argparse).

## Usage#

### Command Line Usage#

```bash
# Get competitive analysis template
python scripts/competitive_analyzer.py --product "vacuum cup"

# Get competitor image generation solution deconstruction
python scripts/image_solution_deconstructor.py --competitor "Thermos"
```
