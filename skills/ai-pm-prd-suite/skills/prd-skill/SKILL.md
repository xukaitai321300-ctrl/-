# 内部 Skill：PRD 生成

> 此 Skill 仅供 Sub Agent 内部调用，不直接暴露给用户

---

## 功能

整合所有讨论，生成最终 PRD。

---

## 输入

```json
{
  "product": "AI 客服助手",
  "decisions": {
    "pain_point": "客服响应慢",
    "ai_suitability": "适合",
    "boundary": {...},
    "confidence": {...},
    "hallucination": {...}
  }
}
```

---

## 输出

根据前面的所有决策，生成完整 PRD 文档，包含：

1. 产品定位
2. AI 能力边界声明
3. 用户交互设计
4. 置信度机制
5. 幻觉应对策略
6. 验收标准
7. 人类决策锚点

---

## 确认流程

生成后需要用户确认：
- "这是我们的讨论结果，你确认一下？"
- "有没有需要调整的？"
