# 内部 Skill：幻觉应对

> 此 Skill 仅供 Sub Agent 内部调用，不直接暴露给用户

---

## 功能

根据用户输入，规划幻觉应对方案。

---

## 输入

```json
{
  "product": "AI 客服助手",
  "tolerance": {
    "fact_error": "zero",
    "logic_error": "low",
    "over_extend": "medium"
  },
  "prevention": ["引用溯源", "回答限制"],
  "remedy": ["立即响应", "提供修正", "用户安抚"]
}
```

---

## 输出

```markdown
# 幻觉应对策略

## 幻觉容忍度
| 类型 | 容忍度 | 原因 |
|------|--------|------|
| 事实错误 | 零容忍 | 可能误导用户 |
| 逻辑错误 | 低容忍 | 需快速修复 |
| 过度延伸 | 可接受 | 危害较小 |

## 预防机制
1. 引用溯源：每句话有来源
2. 回答限制：只回答知识库内容

## 补救方案
1. 立即响应：显示"可能有误"
2. 提供修正：正确答案+原因
3. 用户安抚：道歉+补偿

## 责任边界
- AI 提供错误信息 → 平台责任
- 用户未核实 → 共同责任
```
