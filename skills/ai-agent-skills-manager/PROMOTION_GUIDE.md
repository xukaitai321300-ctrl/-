# 🚀 AI Agent Skills Manager - 推广指南

## 项目概述

**AI Agent Skills Manager** 是一个用于安装和管理 OpenClaw AI Agent 技能的工具。它基于 pskoett/pskoett-ai-skills 项目，提供交互式技能选择和批量安装功能。

## 核心价值

- 🎯 **降低门槛** - 一键安装，无需复杂配置
- 📦 **开箱即用** - 自动克隆、自动安装
- 🎨 **友好界面** - 图形化选择，易于上手
- 🔄 **易于推广** - 单文件分发，零依赖

## 常见问题

### Q: 是否需要 root 权限？
A: 不需要，所有操作在用户级完成。

### Q: 如何卸载？
A: 删除 `~/.openclaw/skills/` 目录中的技能文件夹。

### Q: 支持哪些技能？
A: 当前支持 5 个核心技能，可轻松扩展。

### Q: 如何贡献？
A: 欢迎提交 PR，添加新功能或修复 Bug。
