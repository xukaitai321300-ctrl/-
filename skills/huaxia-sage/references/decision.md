# 决策类框架

## 阴阳平衡 — 双面天平

### 核心理念

阴阳不是"两边都好"，而是"任何事物都有你看不到的另一面"。人天生会偏向自己倾向的那一面，放大优势、缩小隐患。这个框架的作用是**强制把你的视线拉回来**，看另一面。

不是让你选A或选B，而是帮你找到一个"太极点"——融合两面的第三条路。

### 输出结构

```
⚖️ 阴阳天平

你看到的（阳面）：
[用户倾向的选项，列2-3个真实优势]

你忽略的（阴面）：
[用户倾向的选项，列2-3个真实隐患——这些是用户选择性忽视的]

对面的阳面：
[另一选项的真实优势——不用客气，直说]

对面的阴面：
[另一选项的真实代价]

偏离度：X/10
[0=完全中立，10=极端偏向。告诉用户当前偏了多少]

太极点（融合方案）：
[不是二选一，而是一个融合两面的行动方案。
关键是让两个选项在"做"中自然显化，而不是在"想"中纠结]

📜 老祖宗原话
[选一句最贴切的，附一句现代解读]

💡 口诀
[四句押韵，能记住]
```

### 原话储备

- "一阴一阳之谓道。偏阴偏阳之谓疾。" — 《易传》
- "反者道之动，弱者道之用。" — 《老子》
- "祸兮福之所倚，福兮祸之所伏。" — 《老子》

### 示例

用户：用阴阳平衡分析，我该跳槽还是留下

⚖️ **阴阳天平**

**你看到的（阳面）**：
- 新机会薪资确实更高（涨30%）
- 现在的组已经没什么新东西学了

**你忽略的（阴面）**：
- 新公司的涨薪可能是用加班换的（你还没问清楚工作强度）
- "没什么新东西学"可能是因为你还没主动要过新项目
- 跳槽后前3个月基本是消耗期，不是产出期

**对面的阳面**：
- 留下来有信任积累，做什么都更容易推动
- 换组/转岗也是一种"跳"，但风险低得多

**对面的阴面**：
- 留下来如果不主动求变，半年后还是会纠结
- 舒适区待久了，真到不得不走的时候议价权更弱

**偏离度：7/10**（你明显偏向跳槽，已经在心里做了决定）

**太极点**：
不急着跳。用2个月做两件事：
1. 在现司主动找一个有挑战的新方向，测试"留下"的上限
2. 拿到新公司的 offer（不是猎头画的饼），把跳槽从"想法"变成"数据"
两个月后，哪个选项的事实更清晰，就选哪个。

📜 **老祖宗原话**
"一阴一阳之谓道。偏阴偏阳之谓疾。"
—— 极端的选项往往不可持续，真正的道在阴阳平衡处。

💡 **口诀**
不跳不留，边走边看。
阴阳平衡，自然显化。

## 中庸之道 — 刻度尺

### 核心理念

中庸不是"和稀泥"，不是"差不多就行了"。中庸是找到那个**刚刚好的点**——不多不少，不偏不倚。

人要么不做，要么做过头。中庸帮你看见自己在刻度尺上的位置，然后微调。

### 输出结构

```
📏 刻度尺

左极端（过少）：
[具体描述：如果什么都不做/做得太少会怎样]

右极端（过多）：
[具体描述：如果all in/做得太多会怎样]

你当前在：X/10
[标注用户在刻度尺上的位置，0=左极端，10=右极端]

刚刚好（中庸点）：
[具体描述"刚刚好"是什么状态，给一个可感知的标准]

收一步：
[如果你偏右（做得太多），建议做什么来收回一步]

进一步：
[如果你偏左（做得太少），建议做什么来推进一步]

📜 老祖宗原话
[选一句最贴切的]

💡 口诀
[四句押韵]
```

### 原话储备

- "过犹不及。" — 《论语》
- "中庸之为德也，其至矣乎！" — 《论语》
- "执两用中。" — 《中庸》

## 先为不可胜 — 风险优先矩阵

### 核心理念

孙子说"先为不可胜，以待敌之可胜"。意思是：**先想怎么不输，再想怎么赢**。

大多数人做决策是反过来的——先看到赢面就兴奋，然后才想到风险。这个框架倒转思考顺序：先锁死底线，再考虑上行空间。

### 输出结构

```
🛡️ 不可胜分析

先想输法：
├─ 最坏情况是什么？
│  [具体描述，不要模糊]
├─ 什么会导致最坏情况？
│  [列出2-3个触发条件]
└─ 怎么排除这个可能性？
   [针对性的预防措施]

再想赢法：
├─ 排除最坏情况后，剩下什么？
│  [去掉最坏情况后，你的真实处境]
├─ 胜算有多大？
│  [给一个现实的评估，不要灌鸡汤]
└─ 怎么提高胜算？
   [1-2个能提高赢面的小动作]

底线决策：
[即使赢不了，这样做也不亏——这是你的安全网]

信号灯：
🟢 可以上：[满足什么条件就果断上]
🔴 别上：[出现什么信号就立刻撤]

📜 孙子原话
[选一句]

💡 口诀
[四句押韵]
```

### 原话储备

- "善战者，立于不败之地，而不失敌之败也。" — 《孙子兵法》
- "先为不可胜，以待敌之可胜。不可胜在己，可胜在敌。" — 《孙子兵法》
- "胜兵先胜而后求战，败兵先战而后求胜。" — 《孙子兵法》

## 知己知彼 — 全景情报板

### 核心理念

孙子说"知己知彼，百战不殆"。做任何竞争性决策之前，你需要同时看清自己和对手。大多数人只看自己（我想干什么），或者只看对手（他们在干什么），但很少同时看清两边。

这个框架帮你建一个**全景情报板**：你的底牌、对方的底牌、双方的真实差距。

### 输出结构

```
🗺️ 全景情报

知己（你的真实情况）：
├─ 你的优势：[别人没有的、你做得好的]
├─ 你的劣势：[你的短板、别人能看到但你不愿承认的]
├─ 你的底牌：[别人不知道的资源/能力]
└─ 你的盲区：[你可能没意识到的弱点]

知彼（对方的真实情况）：
├─ 对方的优势：[真正让你有压力的地方]
├─ 对方的劣势：[看起来强但实际脆弱的地方]
├─ 对方的意图：[对方真正想要什么——不是他说的]
└─ 对方的盲区：[对方看不到或低估的地方]

实力对比：
[用简洁的方式呈现双方的真实差距——
不是全面对比，而是关键维度的对比]

你的差异化策略：
├─ 避开对方的强项：[不要硬碰的地方]
├─ 打击对方的盲区：[对手看不到/低估的地方，这是你的突破口]
└─ 放大你的独特性：[你最不可替代的地方]

不可忽视的变数：
[什么因素可能改变局面——
比如：对方的策略调整、外部环境变化]

📜 孙子原话
[选一句]

💡 口诀
[四句押韵]
```

### 原话储备

- "知己知彼，百战不殆；不知彼而知己，一胜一负；不知彼，不知己，每战必殆。" — 《孙子兵法》
- "故上兵伐谋，其次伐交，其次伐兵，其下攻城。" — 《孙子兵法》
- "兵者，诡道也。" — 《孙子兵法》
- "凡战者，以正合，以奇胜。" — 《孙子兵法》
