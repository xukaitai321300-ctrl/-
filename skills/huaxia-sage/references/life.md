## 上善若水 — 适应力路线

### 核心理念

老子说"上善若水"——最高的善像水一样。水有三个特质：它遇到石头会绕过去（不硬碰）、它总能找到出路（不绝望）、它慢慢改变地形（不着急）。

不是"顺其自然"的躺平，而是**像水一样找到阻力最小的路径**。水不是软弱，水是最有适应力的东西。

### 输出结构

```
🌊 水的路径

你面对的"石头"（障碍）：
[具体障碍——是什么在挡你]

水的三种策略：
├─ 绕：[能不能绕过去？
  不是逃避，是"此路不通换一条路"]
├─ 渗：[能不能慢慢渗透改变？
  不是正面突破，是"润物细无声"]
└─ 积：[能不能先积蓄力量，等时机？
  不是放弃，是"蓄势待发"]

推荐路径：
[哪种策略适合当前情况——
并解释为什么]

水的特质提醒：
├─ 水不跟石头硬碰 → 你也不需要
├─ 水总能找到出路 → 你也能
└─ 水改变地形 → 时间在你这边

此刻的行动：
[像水一样，最小阻力的下一步——
具体、可执行]

📜 老子原话
[选一句]

💡 口诀
[四句押韵]
```

### 原话储备

- "上善若水。水善利万物而不争。" — 《老子》
- "天下莫柔弱于水，而攻坚强者莫之能胜。" — 《老子》
- "柔弱胜刚强。" — 《老子》
- "江海所以能为百谷王者，以其善下之。" — 《老子》

## 放下执着 — 控制圈

### 核心理念

"放下"不是"别想了"这种废话。放不下的本质是**搞混了能改的和该接的**——把精力花在控制不了的事上，然后因为控制不了而焦虑。

这个框架用一个简单的"控制圈"帮你做区分：你能改变的（内圈）和影响不了的（外圈）。把精力从外圈收回来，集中到内圈。

### 输出结构

```
⭕ 控制圈

能改变的（内圈）：
[你真的能控制的事——
具体、可行动、完全取决于你自己]

影响不了的（外圈）：
[你控制不了的事——
别人的想法、过去发生的事、大环境的变化]

你以为能控制但其实控制不了的：
[帮用户识别"控制幻觉"——
比如"只要我更努力他就会回心转意"
"只要我把方案做得更好老板就会同意"]

你的精力现在花在哪：
[分析用户当前精力分配——
多少在内圈？多少在外圈？]

建议重分配：
├─ 把精力从外圈收回来：[停止做什么]
└─ 集中在内圈的：[开始做什么]

接受练习：
[一个帮助用户练习接受的具体方法——
不是冥想打坐，而是可操作的日常练习，比如"每天花5分钟写一件你无法控制但一直在纠结的事，然后划掉它"]

📜 佛家/道家典故
[选一个贴切的]

💡 口诀
[四句押韵]
```

### 原话储备

- "随遇而安。" — 禅宗
- "菩提本无树，明镜亦非台。本来无一物，何处惹尘埃。" — 六祖惠能
- "放下屠刀，立地成佛。" — 佛家
- "知其不可奈何而安之若命。" — 《庄子》

## 慢工出细活 — 节奏审计

### 核心理念

不是所有事都该快。有些事快了会废（创意、设计、关系修复），有些事就该快（执行、交付、决策）。

这个框架帮你做一次**节奏审计**：你想加快的那件事，到底属于哪一类？如果加速，具体会损失什么？

### 输出结构

```
⏱️ 节奏审计

你想加快的事：
[用户想快的事情]

这件事的类型：
├─ 效率型（越快越好）：
  [如果属于这一类，确实该快]
├─ 质量型（快了会废）：
  [如果属于这一类，加速的代价是什么]
└─ 关系型（急了会伤）：
  [如果属于这一类，着急可能破坏什么]

如果加速的代价：
[具体会损失什么——
质量下降？信任受损？效果打折？]

适当的节奏：
[这件事的理想速度是什么样的——
给一个可感知的描述]

可以并行的：
[在慢的事情进行时，可以同时做什么快的事——
这样你就不会因为"等待"而焦虑]

耐心提示：
[为什么这件事值得等——
不是鸡汤，而是具体的理由]

📜 古人典故
[选一个贴切的]

💡 口诀
[四句押韵]
```

### 原话储备

- "欲速则不达，见小利则大事不成。" — 《论语》
- "合抱之木，生于毫末；九层之台，起于累土。" — 《老子》
- "图难于其易，为大于其细。" — 《老子》
- "好事多磨。" — 民间智慧

## 塞翁失马 — 认知翻转

### 核心理念

塞翁失马的故事人人知道，但真到了自己身上就忘了。坏事不一定是坏事，好事也不一定是好事。关键不是事情本身，而是**你怎么看待这件事，以及你之后做了什么**。

这个框架帮你做一次"认知翻转"：把你眼里的坏事，翻过来看另一面。

### 输出结构

```
🔄 认知翻转

你经历的"坏事"：
[用户描述的 setback / loss / 失败]

直接损失（承认它）：
[不鸡汤，先承认损失是真实的——
具体损失是什么，不要美化]

隐藏收益（翻转它）：
├─ 这个经历教会了你什么：[具体教训]
├─ 它帮你避免了什么：[如果不发生这事，你可能犯更大的错]
└─ 它打开了什么新可能：[因为这件事，你获得了什么新机会/新认知]

历史类比：
[找一个历史上类似的"坏事变好事"的例子——
比如：苏轼被贬黄州反而写出最好的作品]

半年后的你回头看：
[半年后你大概率会怎么看这件事——
大多数"灾难"在半年后都变成了"转折点"]

此刻该做的：
[不是"看开"，而是"利用"——
怎么把这件事变成你的养分]

📜 古人典故
[选一句]

💡 口诀
[四句押韵]
```

### 原话储备

- "祸兮福之所倚，福兮祸之所伏。" — 《老子》
- "塞翁失马，焉知非福。" — 《淮南子》
- "故福之为祸，祸之为福，化不可极，深不可测也。" — 《淮南子》

## 一张一弛 — 精力管理盘

### 核心理念

《礼记》说"一张一弛，文武之道也"。弓不能一直拉着，拉久了弓弦会断。人也不能一直紧绷，绷久了身体会垮。

这个框架帮你做一次**精力审计**：你现在是绷太紧了还是放太松了？你的精力分配是否可持续？

### 输出结构

```
🔋 精力盘

你的精力账户：
├─ 总量评估：[你现在的精力状态——1-10分]
├─ 消耗项：[哪些事在大量消耗你的精力]
└─ 充电项：[哪些事能让你恢复精力——
    注意：刷手机不一定是充电，可能是另一种消耗]

张（紧绷的部分）：
[你一直在"拉弓"的事——
工作、焦虑、内耗、连续高压]

弛（放松的部分）：
[你真正放松的事——
不是"躺尸刷手机"，而是真正让你恢复的活动]

当前状态诊断：
├─ 过张（绷太紧）：[症状 + 风险——
    如：效率下降、情绪暴躁、睡眠差、身体报警]
├─ 过弛（放太松）：[症状 + 风险——
    如：拖延、空虚、自我怀疑]
└─ 刚刚好：[张弛有度的理想状态描述]

节奏调整方案：
[具体的张弛安排——
不是"多休息"这种空话，而是具体到每天/每周的节奏设计]
├─ 每日张弛：[一天中什么时候该紧，什么时候该松]
└─ 每周张弛：[一周中该有至少一次深度恢复]

信号提醒：
[你的身体/情绪会在什么时候给你信号——
学会识别这些信号，在断弦之前主动松弓]

📜 《礼记》原话
[选一句]

💡 口诀
[四句押韵]
```

### 原话储备

- "一张一弛，文武之道也。" — 《礼记》
- "文武之道，未坠于地，在人。" — 《论语》
- "张而不弛，文武弗能也；弛而不张，文武弗为也。" — 《礼记》
- "善养生者，若牧羊然，视其后者而鞭之。" — 《庄子》
