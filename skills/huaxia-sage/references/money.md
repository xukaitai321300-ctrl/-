## 赏罚二柄 — 激励配比方案

### 核心理念

韩非子说"明主之所导制其臣者，二柄而已矣。二柄者，刑德也。"——领导者控制下属只有两个把柄：赏和罚。

关键不是"胡萝卜加大棒"这么简单，而是**精确设计赏罚的比例、时机和力度**。赏太重则人贪婪，罚太重则人恐惧，赏罚不明则人心涣散。

### 输出结构

```
🎣 赏罚设计

目标行为：
[你希望促成或改变什么——先明确目标，再设计激励]

赏（正向激励）：
├─ 是什么：[具体的奖励]
├─ 什么时候给：[时机——即时 vs 延迟]
└─ 力度：[多大合适]

罚（负向约束）：
├─ 是什么：[具体的约束/惩罚]
├─ 什么时候触发：[明确的红线]
└─ 力度：[多大合适]

赏罚比：建议 X:Y
[偏赏还是偏罚，取决于场景——
带团队一般 7:3 偏赏；自我约束可以 5:5；纠正坏习惯可能需要 3:7 偏罚]

常见误区：
[1-2个这个场景下容易犯的激励错误——
比如：赏得太频繁导致贬值、罚得太模糊导致人心惶惶]

如果用于自我激励：
[把赏罚机制用在自己身上——
给自己设定明确的奖励和惩罚，像管理团队一样管理自己]

📜 韩非子原话
[选一句]

💡 口诀
[四句押韵]
```

### 原话储备

- "明主之所导制其臣者，二柄而已矣。二柄者，刑德也。" — 《韩非子》
- "赏莫如厚而信，使民利之；罚莫如重而必，使民畏之。" — 《韩非子》
- "赏厚而信，罚重而必。" — 《韩非子》

## 法术势 — 杠杆分析图

### 核心理念

韩非子集法家大成，提出三个权力要素：
- **法**：规则、制度、流程——谁定的？对谁有利？
- **术**：手段、策略、信息差——各方在用什么手段？
- **势**：趋势、位置、资源——势在哪边？谁是顺势的？

大多数人只知道"术"（用什么手段），但不知道"法"和"势"才是更根本的杠杆。这个框架帮你**看清权力/影响力的真实结构**，找到四两拨千斤的支点。

### 输出结构

```
🏗️ 杠杆分析

法（规则/制度）：
[这个场景下，什么规则在起作用？]
[谁定的规则？规则对谁有利？]
[你能不能改变规则？]

术（手段/策略）：
[各方在用什么手段？]
[哪些手段有效？哪些是虚张声势？]
[你有什么信息差可以利用？]

势（趋势/位置）：
[势在哪边？谁是顺势的？谁是逆势的？]
[大趋势是什么？你在趋势的哪一端？]

你的杠杆点：
├─ 能借的势：[不需要费力就能利用的趋势/资源]
├─ 能用的术：[你手上的有效手段]
└─ 能改的法：[你能影响或制定的规则]

最小阻力路径：
[用最小的力气达成目标的路径——
不是硬碰硬，而是借势用术]

避坑：
[硬碰硬会输的地方——这些地方不要碰]

📜 韩非子原话
[选一句]

💡 口诀
[四句押韵]
```

### 原话储备

- "君无术则弊于上，臣无法则乱于下。" — 《韩非子》
- "势者，胜众之资也。" — 《韩非子》
- "飞龙乘云，腾蛇游雾。" — 《韩非子》（有势则易，无势则难）

## 吃亏是福 — 长期账本

### 核心理念

"吃亏是福"不是让你当冤大头，而是帮你**真的算一笔账**：短期的"亏"能不能换回长期的"福"？如果能，那不是吃亏，是投资。如果不能，那就是真的吃亏了。

关键是用数据说话，别用鸡汤。

### 输出结构

```
📊 长期账本

短期损失（你吃亏的部分）：
[具体损失 + 金额/时间/精力估算]

长期收益（这笔"亏"可能换来的）：
├─ 信任积累：[具体描述]
├─ 声誉资产：[具体描述]
├─ 未来机会：[具体描述]
└─ 关系投资：[具体描述]

算总账：
├─ 如果只看短期：亏了 X
└─ 如果拉长到 1-3 年：可能赚 Y（Y = X 的 N 倍）

这笔亏值不值：
├─ 值得的情况：[什么条件下吃亏是投资——
  比如对方是有长期合作价值的人、吃亏换来的是不可替代的信任]
└─ 不值得的情况：[什么条件下吃亏就是吃亏——
  比如对方没有回报的能力/意愿、吃亏没有换来任何实际资产]

建议：
[基于以上分析的具体建议——
值得就吃，不值得就别勉强当好人]

📜 古人典故
[选一个贴切的]

💡 口诀
[四句押韵]
```

### 原话储备

- "将欲取之，必固与之。" — 《老子》
- "大智若愚，大巧若拙。" — 《老子》
- "塞翁失马，焉知非福。" — 《淮南子》
- "吃亏是福。" — 民间智慧
