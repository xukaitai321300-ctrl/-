## 每日三省 — 三问复盘卡片

### 核心理念

曾子说"吾日三省吾身"——每天从三个维度检查自己。不是自我批评，是**定期校准方向**。

三个问题：为谁谋（你的目标还在吗）、与谁交（你花时间在对的人身上吗）、传谁习（你今天学到了什么值得传递的）。

### 输出结构

```
📝 今日三省

为谁谋？
[今天做的事，最终服务谁的利益？跟你的长期目标一致吗？
如果偏离了，偏离在哪？]

与谁交？
[今天花时间最多的人/事，值得吗？
有没有把时间花在消耗你的人身上？]

传谁习？
[今天学到/发现了什么值得记住的？
有没有什么经验可以传递给他人？]

明天校准：
继续做：[值得保持的]
停止做：[应该放手的]
开始做：[明天可以开始的一个小动作]

📜 曾子原话
[选一句]

💡 口诀
[四句押韵]
```

### 原话储备

- "吾日三省吾身：为人谋而不忠乎？与朋友交而不信乎？传不习乎？" — 《论语》
- "见贤思齐焉，见不贤而内自省也。" — 《论语》
- "过则勿惮改。" — 《论语》

### 用法说明

这个框架可以按两种模式使用：

**模式一：日终复盘**
用户提供今天做了什么，框架输出三省卡片。

**模式二：习惯养成**
用户说"帮我养成XX习惯"，用每日三省的结构设计一个每日自检流程，把目标习惯嵌入到三省中。

## 平常心 — 视角缩小术

### 核心理念

"平常心"不是"别紧张"这种废话。焦虑的本质是**视角放错了焦距**——把一件具体的事放大成了人生的全部。

这个框架用"视角缩小"来解除焦虑：不是让你假装不在乎，而是帮你看到这件事的真实比例。

### 输出结构

```
🔭 视角缩小

你当前的视角（放大镜）：
[你现在把这件事看成了多大的事？描述你的焦虑]

缩小一级（一年后）：
[一年后回头看这件事，你会怎么想？
一年后的你大概率会说什么？]

缩小两级（别人身上）：
[如果这事发生在你的朋友身上，你会怎么劝他？
你会发现：你对自己比对别人苛刻得多]

缩小三级（宇宙视角）：
[100年后，这件事还存在吗？
这个视角不是要你"看开"，而是帮你校准真实的严重程度]

实际影响评估：
[去掉脑补的灾难场景后，最真实的后果是什么？
90%的焦虑来自脑补，不是事实]

此刻能做的：
[回到当下——此刻唯一该做的一件事是什么？
不是想，是做]

📜 禅宗/道家典故
[选一个贴切的故事或原话]

💡 口诀
[四句押韵]
```

### 原话储备

- "宠辱不惊，闲看庭前花开花落。" — 《菜根谭》
- "菩提本无树，明镜亦非台。" — 六祖惠能
- "平常心是道。" — 南泉普愿

## 治未病 — 预警雷达

### 核心理念

《黄帝内经》说"上医治未病"——最高明的医生在病还没发作时就治。大多数灾难不是突然降临的，而是有早期信号的，只是你选择性地忽略了。

这个框架帮你做一次"健康体检"：你现在觉得"还能撑"的小问题，如果不处理，会变成什么？

### 输出结构

```
📡 预警雷达

你提到的小信号：
[用户描述的小问题——把它具象化]

如果不管，它会变成：
├─ 3个月后：[具体的恶化情况]
└─ 1年后：[更严重的后果]

成本对比：
├─ 现在处理（早期）：[时间/精力/金钱]
└─ 以后修复（晚期）：[时间/精力/金钱]
├─ 性价比：现在处理是以后处理的 1/X
└─ [用具体数字说话，不要笼统说"更贵"]

本周就可以做的预防动作：
[1-2个具体的小动作——小到本周就能做，做了就能产生效果]

📜 黄帝内经原话
[选一句]

💡 口诀
[四句押韵]
```

### 原话储备

- "上医治未病，中医治欲病，下医治已病。" — 《黄帝内经》
- "病已成而后药之，乱已成而后治之，譬犹渴而穿井，斗而铸锥，不亦晚乎？" — 《黄帝内经》
- "善战者之胜也，无智名，无勇功。" — 《孙子兵法》（提前消弭危机，所以看起来没什么了不起的）

## 知行合一 — 行动校准器

### 核心理念

王阳明说"知而不行，只是未知"。你以为你"知道"，但你不做，说明你其实不知道——至少不是真知道。

这个框架帮你做一个**知行差距扫描**：你嘴上说的重要的事，你的行为跟上去了吗？如果没跟上，要么你不懂，要么你不够想。

### 输出结构

```
🎯 知行校准

你说的（知的部分）：
[用户声称的目标、价值观、计划]

你做的（行的部分）：
[用户实际的行为模式——从描述中提取]

知行差距：
[说的和做的之间，差了什么？]

差距原因诊断：
├─ 假知（其实不懂）：[你以为你懂了，但你没懂的部分]
├─ 假想（其实不够想）：[你以为你想要，但你内心有抵触的部分]
└─ 假能（其实做不到）：[你知道也想要，但能力/资源/环境不允许的部分]

校准方案：
[针对诊断结果，给一个具体的行动方案——
不是"你应该更努力"，而是拆到最小的可执行动作]

真知验证：
[怎么判断你真的知道了？——
只有一个标准：你做到了]

📜 王阳明原话
[选一句]

💡 口诀
[四句押韵]
```

### 原话储备

- "知而不行，只是未知。" — 王阳明
- "未有知而不行者。知而不行，只是未知。" — 王阳明
- "知是行的主意，行是知的功夫。" — 王阳明
- "知之真切笃实处即是行，行之明觉精察处即是知。" — 王阳明

## 格物致知 — 深度学习法

### 核心理念

《大学》说"致知在格物"——想真正搞懂一件事，就去深入研究它的本质。不是泛读100篇文章，而是把一件事拆到底、钻到透。

现代人学东西太"浅"：收藏了一堆文章、听了很多课、觉得"我了解了"，但真正让他讲清楚就说不出。格物致知帮你从"知道名词"到"理解本质"。

### 输出结构

```
🔬 格物路线

你想搞懂的事：
[用户想学的新领域/新技能/新知识]

它的"物"（本质）是什么：
[这件事最底层的原理/逻辑是什么——
不是表面的操作步骤，而是"为什么会这样"]

从外到内的拆解：
├─ 第一层（表象）：[大部分人看到的是什么]
├─ 第二层（逻辑）：[背后的运行规律是什么]
└─ 第三层（本质）：[最核心的1-2个原理是什么]

你的"格"法：
[针对这件事，建议用什么方式深入研究——
比如：亲手做一遍、教给别人、写出来、找源头论文]

学习路径：
Step 1: [第一个该做的事——通常是最底层的原理学习]
Step 2: [第二个——实践验证]
Step 3: [第三个——输出/教别人，检验是否真懂]

避坑：
[学这个东西最常见的误区——
比如：看太多教程不动手、追求广度不追求深度]

📜 《大学》原话
[选一句]

💡 口诀
[四句押韵]
```

### 原话储备

- "致知在格物，物格而后知至。" — 《大学》
- "格物致知。" — 《礼记·大学》
- "知之真切笃实处即是行，行之明觉精察处即是知。" — 王阳明
- "学而不思则罔，思而不学则殆。" — 《论语》
