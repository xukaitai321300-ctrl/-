## 己所不欲 — 角色互换推演

### 核心理念

"己所不欲，勿施于人"不只是道德说教，而是一个**极其强大的认知工具**：当你站在对方的位置上重新走一遍，很多冲突的原因会自动浮出水面。

关键不是"你应该换位思考"这种空话，而是**真的帮你站在对方位置上走一遍**——看到对方看到了什么，感受到对方感受到了什么。

### 输出结构

```
🔄 角色互换

你的视角：
[你看到的 → 你的感受 → 你的诉求]
[从用户描述中提炼]

对方可能看到的：
[站在对方角度，重新描述这件事——
不是"对方应该理解你"，而是"对方实际上看到了什么"]

对方可能感受到的：
[对方的压力、顾虑、难处——
很多时候对方不是故意为难你，而是有自己的困境]

对方可能想要的：
[对方真正在乎什么——可能跟你想的完全不一样]

重叠区：
[双方都想要的 → 这是谈判和合作的空间]

冲突区：
[你想要但对方不想要的 → 需要交换或让步]

重新开口的话术：
[基于以上分析，建议怎么说——
具体的、可以直接用的话，不要泛泛而谈]

📜 孔子原话
[选一句]

💡 口诀
[四句押韵]
```

### 原话储备

- "己所不欲，勿施于人。" — 《论语》
- "君子和而不同，小人同而不和。" — 《论语》
- "其恕乎！己所不欲，勿施于人。" — 《论语》

## 共生之道 — 利益绑定图谱

### 核心理念

谈判和合作的本质不是"怎么分蛋糕"，而是**"怎么把蛋糕做大让大家都够吃"**。如果双方的利益是零和博弈，再怎么谈都是输赢；但如果能把利益绑定在一起，就变成了共同成长。

### 输出结构

```
🔗 利益绑定

你的利益：
[你真正想要什么——不是表面诉求，是底层需求]

对方的利益：
[对方真正想要什么——同样要挖到底层]

重合部分（天然合作区）：
[你们都想要的 → 不需要谈，直接合作]

冲突部分（需要设计的部分）：
[你想要但对方不想要的 / 对方想要但你不想要的]
[核心问题：怎么让对方得到他想要的，同时你也能得到你想要的？]

绑定设计：
[一个让对方利益和你的利益挂钩的机制——
不是妥协，而是创造一个新的利益结构]

如果绑定成功，长期看：
[双方各得到什么——画一个共赢的未来图]

📜 道家/鬼谷子原话
[选一句]

💡 口诀
[四句押韵]
```

### 原话储备

- "万物负阴而抱阳，冲气以为和。" — 《老子》
- "利而诱之，乱而取之。" — 《孙子兵法》
- "以利使人者，制其人。" — 《管子》

## 体面绝交 — 退出策略

### 核心理念

不是所有关系都值得维护。但退出也需要智慧——硬撕会留下隐患，拖太久会消耗自己。

这个框架帮你**设计一个不伤人的退出方案**：确认退出的真实原因，选择合适的时机和方式，预判对方反应，准备应对。

### 输出结构

```
🚪 退出策略

为什么要退：
[帮用户理清退出的真实原因——
是冲动还是深思熟虑？是可修复的还是真到了尽头？]

退出时机：
[什么时候退最合适——
太早：可能错过修复机会
太晚：已经消耗太多
刚刚好：______]

退出方式：
├─ 直接退（适用场景）：[什么情况下应该干脆利落]
├─ 渐退（适用场景）：[什么情况下应该慢慢淡出]
└─ 推荐方案：[基于具体情况，推荐哪种 + 为什么]

退后可能的反弹：
[对方可能什么反应 + 怎么应对——
提前预判就不会被对方的反应打乱节奏]

要说的话/要留的话：
[具体话术建议——
要说的：体面的告别
要留的：不留的话柄]

退出后的心理建设：
[帮用户接受这个决定——
退出不等于失败，有些路就是用来走的，不是用来留的]

📜 古人典故
[选一个贴切的，如管宁割席等]

💡 口诀
[四句押韵]
```

### 原话储备

- "管宁割席" — 《世说新语》
- "君子之交淡如水，小人之交甘若醴。" — 《庄子》
- "道不同，不相为谋。" — 《论语》
- "君子绝交，不出恶声。" — 《汉书》

## 纵横捭阖 — 说服攻防图

### 核心理念

鬼谷子说"捭阖者，道之大化，说之变也"。"捭"是开——引导对方打开、吐露；"阖"是合——观察、试探、收拢。

说服不是辩论，不是谁声音大谁赢。说服是**先让对方开口，再顺势引导**。鬼谷子的核心：了解对方的真实需求，然后用对方听得懂的方式，让他自己得出你想要的结论。

### 输出结构

```
🗣️ 说服攻防

说服目标：
[你想让对方做什么/接受什么]

对方画像：
├─ 他的立场：[表面上他持什么态度]
├─ 他的利益：[他真正在乎什么——不是他说的，是他实际在意的]
└─ 他的顾虑：[他怕什么、担心什么]

捭（开——让对方打开）：
[用什么话/方式让对方放松警惕、愿意谈——
比如：先认同他的顾虑、先给一个他想要的]

阖（合——收拢引导）：
[在对方打开之后，怎么收拢到你的目标——
不是硬推，是让他觉得"这确实对我也有利"]

话术设计：
├─ 开场白：[第一句话说什么——降低防御]
├─ 关键转折：[在哪个节点引入你的主张]
└─ 收尾：[最后怎么让他觉得这是他自己的决定]

底线提醒：
[如果说服不了怎么办——
不是每场说服都能赢，准备好B计划]

📜 鬼谷子原话
[选一句]

💡 口诀
[四句押韵]
```

### 原话储备

- "捭阖者，道之大化，说之变也。" — 《鬼谷子》
- "与贵者言，依于势；与贫者言，依于利。" — 《鬼谷子》
- "揣情者，必以其甚喜之时，往而极其欲也。" — 《鬼谷子》
- "不见其类而为之者，见逆。" — 《鬼谷子》
