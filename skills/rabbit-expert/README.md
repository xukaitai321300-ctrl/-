# Rabbit Expert Skill Package (十二生肖团 - 兔专家)#

兔专家（图像分析专家）技能包，负责分析生成图质量、特征提取。包含生成图质量分析、特征提取、图像对比分析等功能。是十二生肖团的图像分析者。

## Features#

- **Generated Image Quality Analysis**: Clarity, color accuracy, composition reasonableness
- **Feature Extraction**: Edge detection, texture analysis, color histogram
- **Image Comparason Analysis**: Comparison with competitor images, comparison with reference images#

## Installation#

1. Ensure Python 3.8+ is installed.
2. Install dependencies: `pip install pillow opencv-python`#

## Usage#

```bash
# Analyze generated image quality
python scripts/image_quality_analyzer.py --image "path/to/image.jpg"

# Extract image features
python scripts/feature_extractor.py --image "path/to/image.jpg"
```
