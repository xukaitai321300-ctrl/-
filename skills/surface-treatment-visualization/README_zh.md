# 表面处理可视化技能包

提供各种表面处理技术的可视化功能，包括喷漆、喷塑、热转印、激光雕刻等效果的展示和对比。

## 功能描述

本技能包提供各种表面处理技术的可视化功能，包括：
- **喷漆效果可视化**: 展示不同漆面效果（高光、哑光、金属漆等）
- **喷塑效果可视化**: 展示塑料喷涂效果（纹理、厚度、颜色等）
- **热转印效果可视化**: 展示热转印图案和纹理效果
- **激光雕刻效果可视化**: 展示激光雕刻深度、精度和图案效果
- **表面处理前后对比**: 展示表面处理前后的效果对比
- **表面处理工艺参数可视化**: 展示温度、时间、厚度等工艺参数对效果的影响

## 核心功能

### 1. 表面处理效果生成
- 生成各种表面处理效果的图片
- 支持自定义参数（颜色、纹理、厚度等）
- 支持批量生成和对比

### 2. 表面处理效果展示
- 展示表面处理效果图片
- 支持放大、缩小、旋转等操作
- 支持效果对比和切换

### 3. 表面处理前后对比
- 生成表面处理前后的对比图片
- 支持滑块对比和分屏对比
- 支持标注和说明

### 4. 表面处理工艺参数可视化
- 可视化工艺参数对效果的影响
- 生成参数-效果关系图
- 支持参数优化和建议

## 工作流程

### 阶段1：需求分析
1. 明确表面处理类型（喷漆、喷塑、热转印、激光雕刻等）
2. 确定可视化目标（效果展示、前后对比、参数可视化等）
3. 确定输出格式（图片、图表、报告等）

### 阶段2：参数设置
1. 设置表面处理参数（颜色、纹理、厚度、温度、时间等）
2. 设置可视化参数（分辨率、视角、光照等）
3. 设置输出参数（格式、尺寸、质量等）

### 阶段3：效果生成
1. 生成表面处理效果图片
2. 生成前后对比图片
3. 生成工艺参数可视化图表

### 阶段4：效果展示和评估
1. 展示生成的效果图片和图表
2. 评估效果质量和准确性
3. 优化参数和重新生成

## 安装和使用

```bash
pip install numpy matplotlib PIL opencv-python
```

### 使用示例

#### 示例1：生成喷漆效果
```python
from scripts.effect_generator import EffectGenerator

# 初始化效果生成器
generator = EffectGenerator()

# 生成喷漆效果
effect = generator.generate_painting_effect(
    color="红色",
    finish="高光",
    material="镁合金",
    output_path="output/painting_effect.png"
)

# 输出效果图片路径
print(f"喷漆效果图片已生成: {effect['output_path']}")
```

#### 示例2：生成表面处理前后对比
```python
from scripts.comparison_generator import ComparisonGenerator

# 初始化对比生成器
comparator = ComparisonGenerator()

# 生成前后对比
comparison = comparator.generate_before_after_comparison(
    before_image="input/raw_surface.jpg",
    after_effect="喷塑",
    after_params={"color": "蓝色", "texture": "细纹"},
    output_path="output/comparison.png"
)

# 输出对比图片路径
print(f"前后对比图片已生成: {comparison['output_path']}")
```

## 支持的表面处理技术

### 1. 喷漆
- **效果类型**: 高光、哑光、金属漆、珠光漆、锤纹漆
- **参数**: 颜色、光泽度、厚度、层数
- **应用场景**: 金属杯体、塑料外壳、配件

### 2. 喷塑
- **效果类型**: 纹理、颜色、厚度
- **参数**: 颜色、纹理类型、厚度、温度
- **应用场景**: 金属框架、塑料外壳

### 3. 热转印
- **效果类型**: 图案、纹理、颜色
- **参数**: 图案、温度、压力、时间
- **应用场景**: 杯体图案、LOGO、装饰

### 4. 激光雕刻
- **效果类型**: 深度、精度、图案
- **参数**: 深度、精度、速度、功率
- **应用场景**: LOGO雕刻、图案雕刻、文字雕刻

### 5. 阳极氧化
- **效果类型**: 颜色、光泽、厚度
- **参数**: 颜色、厚度、温度、时间
- **应用场景**: 铝合金杯体、配件

### 6. 电镀
- **效果类型**: 颜色、光泽、厚度
- **参数**: 镀层类型、厚度、温度、电流
- **应用场景**: 金属配件、装饰件

## 输出格式

### 1. 效果图片
- **格式**: PNG、JPG、SVG
- **分辨率**: 自定义（默认1920x1080）
- **内容**: 表面处理效果展示

### 2. 前后对比图片
- **格式**: PNG、JPG
- **分辨率**: 自定义（默认1920x1080）
- **内容**: 表面处理前后对比

### 3. 工艺参数可视化图表
- **格式**: PNG、SVG、HTML
- **类型**: 折线图、柱状图、散点图、热力图
- **内容**: 工艺参数对效果的影响

### 4. 效果报告
- **格式**: JSON、HTML、PDF
- **内容**: 效果描述、参数设置、质量评估、优化建议

## 质量标准

### 效果生成质量
- **真实性**: 效果图片的真实感和准确性
- **细节丰富度**: 效果细节的丰富程度
- **参数可控性**: 参数调整的灵活性和精度
- **生成效率**: 效果生成的速度

### 可视化质量
- **清晰度**: 可视化结果的清晰度
- **直观性**: 可视化结果的直观易懂性
- **交互性**: 可视化结果的交互性
- **信息量**: 可视化结果的信息量
