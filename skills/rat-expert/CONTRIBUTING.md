# Contributing to Rat Expert Skill Package

我们欢迎对本技能包的贡献！请阅读以下指南，以确保您的贡献能够顺利地被接受。

## 贡献流程

1. **Fork 本仓库**
2. **创建您的特性分支** (`git checkout -b feature/AmazingFeature`)
3. **提交您的更改** (`git commit -m 'Add some AmazingFeature'`)
4. **推送到分支** (`git push origin feature/AmazingFeature`)
5. **创建一个 Pull Request**

## 代码规范

- 使用 Python 3.8+ 编写代码
- 遵循 PEP 8 代码规范
- 添加适当的注释（中文注释优先）
- 更新相关文档

## 数据规范

- 所有数据文件使用 JSON 格式
- 所有数据条目包含中英文双语字段
- 确保数据准确性和实用性

## 测试规范

- 所有 Python 工具脚本必须有对应的测试文件
- 测试文件使用 `unittest` 或 `pytest` 框架
- 确保测试覆盖率 ≥ 80%

## 文档规范

- 所有文档使用 Markdown 格式
- 所有文档包含中英文双语版本
- 保持文档与代码同步更新

## 问题反馈

如果您发现任何问题或有任何改进建议，请：
1. 查看现有 Issues，确保问题未被报告
2. 创建新的 Issue，详细描述问题和建议
3. 联系：速凡团队（浙江永康）

## 许可证

通过贡献本技能包，您同意您的贡献将在 MIT 许可证下授权。
