# Realistic Photography Skill Package

本技能包提供保温杯产品的真实摄影生图能力，包含摄影灯光模拟、材质纹理增强、产品摄影构图、高清细节处理等功能。专为替代已归档的羊专家（zheng10-ai-image-generator）技能包而创建，是 P0 优先级的核心生图能力技能包。

## Features

- **Lighting Simulation**: Three-point lighting, top light, window light setups
- **Material Texture Enhancement**: Stainless steel, titanium, PP, TRITAN, silicone texture enhancement
- **Product Photography Composition**: Front view, side view, top view, detail close-up, scene matching composition templates
- **High-Resolution Detail Processing**: Material surface detail, LOGO process detail, assembly seam detail, lighting reflection detail, color accuracy detail

## Installation

1. Ensure Python 3.8+ is installed.
2. No additional Python dependencies are required (only standard libraries: json, argparse).

## Usage

### Command Line Usage

```bash
# Get lighting setup configuration
python scripts/lighting_simulator.py --setup three_point

# Get material texture enhancement prompts
python scripts/texture_enhancer.py --material stainless_steel_304

# Get product photography composition template
python scripts/composition_generator.py --template front_view

# Get high-resolution detail processing prompts
python scripts/detail_processor.py --detail material_surface_detail
```

### Using in ComfyUI

1. **Generate Prompts**: Run the above command-line tools to get English/Chinese prompt keywords.
2. **Input Prompts**: Copy the prompts into ComfyUI's Text Prompt node.
3. **Adjust Parameters**: Adjust ComfyUI parameters according to lighting setup and camera settings.
4. **Generate Image**: Run the ComfyUI workflow to generate realistic photography-grade product images.

### Using in AI Image Generation Workflow

The prompts generated by this skill package can be directly used in:
- **ComfyUI**: Text-to-Image, Image-to-Image workflows
- **Stable Diffusion**: Text-to-Image, Image-to-Image
- **Midjourney**: `/imagine` command
- **DALL-E**: Text description to image
