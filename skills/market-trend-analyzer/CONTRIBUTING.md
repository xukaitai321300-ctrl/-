# Contributing Guide

感谢您考虑为市场趋势分析技能包（market-trend-analyzer）做出贡献！

## 如何贡献

### 1. 报告问题

如果您发现问题或建议改进，请通过以下方式联系我们：
- 创建Issue（如果项目有GitHub仓库）
- 直接联系速凡团队

### 2. 提交改进

我们欢迎任何形式的贡献，包括：
- 代码改进
- 文档更新
- 数据更新
- 新功能建议

### 3. 数据贡献

本技能包依赖准确的市场数据。如果您有更新的市场数据，欢迎贡献：
- 更新 `data/market_trends.json`（市场趋势数据）
- 更新 `data/consumer_demand.json`（消费者需求数据）
- 更新 `data/competitors.json`（竞争对手数据）
- 更新 `data/market_forecast.json`（市场预测数据）

### 4. 代码贡献

如果您想贡献代码，请遵循以下 guidelines：
- 使用清晰的 commit 消息
- 添加必要的注释
- 更新相关文档
- 确保代码不影响现有功能

### 5. 测试

贡献代码后，请运行测试以确保功能正常：
```bash
cd E:\\AI日记\\Claw\\skills\\market-trend-analyzer
python scripts/trend_analyzer.py
python scripts/competitor_tracker.py
python scripts/forecast_tool.py
```

## 数据更新频率

- **市场趋势数据**：建议每月更新一次
- **消费者需求数据**：建议每季度更新一次
- **竞争对手数据**：建议每周更新一次
- **市场预测数据**：建议每季度更新一次

请参考 [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) 了解我们的行为准则。
