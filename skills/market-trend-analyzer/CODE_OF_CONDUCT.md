# Code of Conduct

## 我们的承诺

为了营造一个开放、友好的环境，我们作为贡献者和维护者承诺：
- 使参与本项目的人不受骚扰，无论其年龄、身体尺寸、残疾、种族、性别特征、性别认同和表达、经验水平、教育程度、社会身份、国籍、个人外貌、种族、宗教或性认同和性取向。
- 尊重不同的观点和经历。
- 优雅地接受建设性批评。
- 关注对社区最有利的事情。
- 表现出对其他社区成员的同理心。

## 我们的标准

有助于创造积极环境的行为示例包括：
- 使用欢迎和包容的语言。
- 尊重不同的观点。
- 接受建设性批评。
- 关注社区的最佳利益。
- 对其他社区成员表现出同理心。

不可接受的行为示例包括：
- 使用性意味的语言或图像，以及不受欢迎的性关注或挑逗。
- 挑衅、侮辱或贬低性评论，以及个人或政治攻击。
- 公开或私下骚扰。
- 未经明确许可，发布他人的私人信息，如物理地址或电子地址。
- 其他在专业环境中可能被合理认为不当的行为。

## 执行责任

项目维护者负责澄清和执行行为准则的标准，并有权删除、编辑或拒绝与本行为准则不一致的评论、提交、代码、wiki 编辑、问题和其他贡献。

项目维护者有权并有责任删除、编辑或拒绝对本行为准则不遵守的评论、提交、代码、wiki 编辑、问题和其他贡献，或暂时或永久禁止任何贡献者进行其他行为，这些行为他们认为不合适、威胁、冒犯或有害。

## 范围

本行为准则适用于项目空间和公共空间，当个人代表项目或其社区时。示例包括使用官方项目电子邮件地址、通过官方社交媒体账户发布或作为在线或离线事件的指定代表。项目维护者可以另外定义和澄清项目代表。

## 执行

滥用、骚扰或其他不可接受的行为的实例可以通过联系项目维护团队进行报告。所有投诉都将得到审查和调查，并将根据具体情况做出回应，项目维护者认为必要和适当的。项目团队有义务对事件的报告者保密。具体执行政策的更多详细信息可能会单独发布。

不善意遵循或执行行为准则的项目维护者可能会面临由项目领导团队确定的临时或永久性影响。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/)，版本 2.1，可从 https://www.contributor-covenant.org/version/2/1/code_of_conduct/ 获取。

## 联系方式

如有任何问题，请联系速凡团队。

感谢您的合作！
