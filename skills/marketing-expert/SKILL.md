# 市场营销专家（Marketing Expert）技能包

## 功能概述

本技能包提供市场营销功能，包含：

1. **市场营销策略制定** - 市场营销策略制定、目标市场分析、营销预算分配、营销效果预测
2. **品牌推广** - 品牌推广策略制定、品牌推广活动策划、品牌推广效果分析
3. **数字营销** - 数字营销策略制定、社交媒体营销、内容营销、搜索引擎营销
4. **营销效果分析** - 营销效果数据分析、营销效果报告生成、营销优化建议

市场营销专家是十二生肖团的**市场营销专家**，负责市场营销策略制定、品牌推广、数字营销。在十二生肖团工作流中，市场营销专家位于第8阶段（市场营销阶段），接收蛇专家的产品设计方案，进行市场营销策略制定，然后将市场营销方案传递给猪专家（品牌设计）、鸡专家（质量评审）。

## 详细功能说明

### 1. 市场营销策略制定 (Marketing Strategy Formulation)

#### 1.1 市场营销策略制定 (Marketing Strategy Formulation)

**功能**：制定市场营销策略，确保市场营销策略全面、准确。

**输入**：
- `project_name`：项目名称（例如："弹跳盖保温杯"）
- `target_market`：目标市场（例如：["高端市场", "中端市场", "低端市场"]）
- `marketing_budget`：营销预算（例如：100000元）

**输出**：
- `marketing_strategy_report.md`：市场营销策略报告，包含：
  - 市场营销策略概述
  - 目标市场分析
  - 营销预算分配
  - 营销效果预测

**示例**：
```bash
python scripts/marketing_strategy_formulator.py --project "弹跳盖保温杯" --target_market "高端市场,中端市场" --marketing_budget 100000
```

**输出文件示例** (`marketing_strategy_report.md`)：
```markdown
# 市场营销策略报告：弹跳盖保温杯

## 1. 市场营销策略概述
- 项目名称：弹跳盖保温杯
- 营销目标：提升品牌知名度、增加产品销量
- 营销策略：线上线下结合、社交媒体营销、内容营销

## 2. 目标市场分析
- 高端市场：占比30%，目标客户：商务人士、礼品购买者
- 中端市场：占比50%，目标客户：家庭用户、上班族
- 低端市场：占比20%，目标客户：学生、价格敏感者

## 3. 营销预算分配
- 线上营销：60%（社交媒体营销、内容营销、搜索引擎营销）
- 线下营销：40%（商场促销、展会推广、经销商奖励）

## 4. 营销效果预测
- 品牌知名度提升：30%
- 产品销量增加：50%
- 营销投资回报率：200%
```

### 2. 品牌推广 (Brand Promotion)

#### 2.1 品牌推广策略制定 (Brand Promotion Strategy Formulation)

**功能**：制定品牌推广策略，确保品牌推广策略全面、准确。

**输入**：

- `brand_positioning`：品牌定位（例如："高端、轻量化、智能化"）
- `promotion_channels`：推广渠道（例如：["社交媒体", "电商平台", "线下商场"]）

**输出**：
- `brand_promotion_strategy_report.md`：品牌推广策略报告，包含：
  - 品牌推广策略概述
  - 品牌定位分析
  - 推广渠道选择
  - 推广活动策划

**示例**：
```bash
python scripts/brand_promotion_strategy_formulator.py --project "弹跳盖保温杯" --brand_positioning "高端、轻量化、智能化" --promotion_channels "社交媒体,电商平台,线下商场"
```

python scripts/digital_marketing_strategy_formulator.py --project "弹跳盖保温杯" --digital_marketing_channels "微信,微博,抖音,小红书" --digital_marketing_budget 50000

### 4. 营销效果分析 (Marketing Effect Analysis)

#### 4.1 营销效果数据分析 (Marketing Effect Data Analysis)

**功能**：分析营销效果数据，确保营销效果分析全面、准确。

**输入**：
- `project_name`：项目名称（例如："弹跳盖保温杯"）
- `marketing_effect_data`：营销效果数据文件路径（CSV格式）

**输出**：
- `marketing_effect_analysis_report.md`：营销效果分析报告，包含：
  - 营销效果数据分析概述
  - 营销效果数据分析结果
  - 营销效果分析建议

**示例**：
```bash
python scripts/marketing_effect_data_analyzer.py --project "弹跳盖保温杯" --marketing_effect_data "path/to/marketing_effect_data.csv"
```
